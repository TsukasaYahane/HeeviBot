# HeeviBot
基于Nonebot和OneBot协议开发的个人QQ机器人  

推荐使用[Lagrange.Core](https://github.com/LagrangeDev/Lagrange.Core)进行部署  

# 关于本项目  
本项目由个人以学习为目的独立开发维护，欢迎各位提出意见和建议
